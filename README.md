# Alien Invasion(シューティングゲーム)
## 概要
- pygameで作成したシューティングゲームのプログラムです。
- resoucesフォルダに、プログラム作成時に使ったメモやフローチャートなどを置いています。
- programフォルダの中の main.py がシューティングゲームのプログラム本体です。

## Option
- high_scoreを、データベースまたはファイルに保存し、プログラムが実行されたら読み込む
- リファクタリング
